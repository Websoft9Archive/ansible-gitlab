# Akeneo 自动化安装与部署（研发中）

本项目是基于 Ansible 编写的 akeneo 自动化安装脚本，只要熟悉 Ansible，便可以将 akeneo 用无人值守的方式部署到你的服务器。本项目是开源项目，支持MIT开源协议。如果您不熟悉 Ansible 的使用，您可以直接使用我们在公有云上提供的 [相关镜像](https://apps.websoft9.com/akeneo)。

## 操作系统

目前仅支持Ubuntu18.x以上

## 服务器要求

最低2核4G
